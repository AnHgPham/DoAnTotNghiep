% -------- TOPIC --------
\textbf{Topic:} \\[0.15cm]
Enhanced Few-Shot Open-Set Keyword Spotting with Noise-Robust Prototype Classification and Real-Time Streaming Inference
\\[0.15cm]
\hline

% -------- DESCRIPTION --------
\textbf{Description:} \\[0.1cm]
Keyword Spotting (KWS) is the task of detecting predefined spoken keywords in audio streams --- the core technology behind voice-activated systems such as ``Hey Siri'' and ``OK Google.'' Traditional KWS systems require thousands of labeled samples per keyword and can only recognize a fixed vocabulary defined during training.

\vspace{0.15cm}
This project builds a KWS system that overcomes these limitations through:
\begin{itemize}[leftmargin=0.8cm, topsep=1pt, itemsep=1pt, parsep=0pt]
    \item \textbf{Few-Shot Learning:} Recognize new keywords from only 3--5 audio samples without retraining, using Prototypical Networks where each keyword is represented by the average embedding (prototype) of its few samples.
    \item \textbf{Open-Set Recognition:} Reject unknown words using an acceptance-radius threshold around each prototype --- if the L2 distance exceeds this threshold, the input is classified as ``unknown.''
\end{itemize}

\vspace{0.1cm}
The main contribution is a \textbf{Direct L2 distance classification} method replacing probability-based scoring, providing a more intuitive open-set rejection mechanism with a clear geometric meaning.

\vspace{0.15cm}
\textbf{System pipeline:}
\begin{enumerate}[leftmargin=0.8cm, topsep=1pt, itemsep=1pt, parsep=0pt]
    \item Audio preprocessing: raw audio (16\,kHz, 1s) $\rightarrow$ MFCC features (40 computed, 10 retained).
    \item Feature encoding: DSCNN-L (276 channels) $\rightarrow$ 276-dim embeddings, trained with Triplet Loss on MSWC English.
    \item Classification: Open Nearest Class Mean (openNCM) with Direct L2 distance.
    \item Noise robustness: DEMAND noise augmentation (SNR\,=\,5\,dB) + optional denoising at inference.
    \item Streaming inference: Silero VAD + sliding window (1s window, 0.5s stride).
\end{enumerate}

\vspace{0.1cm}
\textbf{Datasets:} Google Speech Commands v2 (evaluation), MSWC English 763 words (training), DEMAND (noise).\\
\textbf{Tools:} Python 3.10+, PyTorch 2.0+, torchaudio, Gradio (demo), Google Colab (GPU training).
\\[0.15cm]
\hline

% -------- OBJECTIVES --------
\textbf{Objectives:} \\[0.1cm]
The tasks include:
\begin{enumerate}[leftmargin=0.8cm, topsep=1pt, itemsep=2pt, parsep=0pt]
    \item Build a fully functional few-shot open-set KWS system capable of learning new keywords from 3--5 audio samples without retraining the neural network.
    \item Implement and evaluate the proposed Direct L2 distance classification method against a probability-based baseline, using DET curves, AUC, and ACC@5\%FAR across multiple protocols (GSC Fixed, GSC Randomized, MSWC Randomized).
    \item Develop a noise robustness module with training-time noise augmentation (DEMAND dataset) and benchmark KWS accuracy at different SNR levels (0, 5, 10, 20\,dB, clean).
    \item Implement a real-time streaming inference module integrating Voice Activity Detection (Silero VAD) and sliding-window keyword detection with end-to-end latency under 100\,ms.
    \item Build an interactive Gradio web demo supporting keyword enrollment, offline detection, and simulated streaming detection.
    \item Deliver complete source code, trained model checkpoints, evaluation results, and a written thesis report.
\end{enumerate}
\\[0.1cm]
